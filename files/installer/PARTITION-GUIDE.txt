║                  Bingux Partition Guide                         ║
║           Recommended UEFI/GPT Layout with LUKS                ║
╚══════════════════════════════════════════════════════════════════╝

This installer uses Calamares to partition your disk. Below is
the recommended partition layout for a UEFI system with full-disk
encryption (LUKS2).

┌─────────────────────────────────────────────────────────────────┐
│ #   Mount point   Size          Format    Flags                │
├─────────────────────────────────────────────────────────────────┤
│ 1   /boot         1024 MiB      fat32     boot                 │
│ 2   [SWAP]        ≥ RAM size    swap      swap                 │
│ 3   /             Remainder     btrfs     (encrypt)            │
└─────────────────────────────────────────────────────────────────┘

PARTITION DETAILS
─────────────────

1. EFI System Partition (/boot)
   • File System: fat32
   • Mount Point: /boot
   • Size: 1024 MiB
   • Flags: tick "boot"
   • Do NOT encrypt this partition.

2. Swap (optional but recommended)
   • File System: swap (linuxswap)
   • Size: at least equal to your RAM for hibernate support.
     8 GiB is a good default for most systems.
   • Flags: tick "swap"
   • Alternatively, skip this and use a swap file later.

3. Root Partition (/)
   • File System: btrfs
   • Mount Point: /
   • Size: use the remainder of the disk.
   • Tick "Encrypt" to enable LUKS2.
   • Calamares will automatically create btrfs subvolumes:
       @           → /
       @home       → /home
       @nix        → /nix
   • Mount options: compress=zstd,noatime (set automatically).

USING "ERASE DISK" (EASY MODE)
──────────────────────────────
If you don't need a custom layout:
1. Select "Erase disk".
2. Tick "Encrypt system" and enter your passphrase.
3. Calamares will auto-create EFI + encrypted root for you.
4. Review the summary and click "Install".

USING "MANUAL PARTITIONING" (ADVANCED)
──────────────────────────────────────
1. Click "Manual partitioning".
2. Select your target disk and click "New Partition Table" → GPT.
3. Create partitions as described in the table above:
   • First: 1024 MiB, fat32, mount /boot, flag "boot".
   • Second (optional): swap partition sized to your RAM.
   • Third: remaining space, btrfs, mount /, tick "Encrypt".
4. Review and install.

ENCRYPTION NOTES
────────────────
• When you tick "Encrypt", Calamares creates a LUKS2 container
  on the root partition and prompts for a passphrase.
• On first boot, Plymouth will display a graphical passphrase
  prompt (the Bingux splash screen).
• Choose a strong passphrase — this protects all data on disk.

For manual install (advanced), see /etc/bingux-installer/README.
